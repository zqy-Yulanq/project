# Development Environment
### Software of Building Machine Learning Prediction Models: Jupyter Notebook
The Jupyer Notebook version used in this project is 6.3.0, which can implement all interactive calculations and document writing in web pages.
### Software of Implementing Housing Price Prediction System：PyCharm
The PyCharm version used in this project is 2021.2.2, which is used to implement the code of the web page.
### Language： Python
The programming language used in Jupyter Notebook and PyCharm is Python, and the version used is Python 3.8.
# Usage
1. The user first needs to have the development environment mentioned above. 
2. Users can click on the Boston_Housing.ipynb and Lanzhou_Second-hand housing.ipynb files to open two web pages, which respectively introduce the method of building machine learning models in this project. Then click Run button to run the entire application. 
3. For the housing price prediction system, you need to first open the web folder in PyCharm, then run the run.py file, and then click the generated link to jump to the web page of the housing price prediction system. After that, the user needs to register and login to use the system.
